# objetivo del proyecto 
Este proyecto consiste en un análisis exploratorio de datos (EDA) de clientes de la empresa ConectaTel. El objetivo principal es evaluar la calidad de los datos, identificar problemas como valores nulos, sentinels y fechas inválidas, analizar el comportamiento de los usuarios y segmentarlos según su edad y nivel de uso.

A partir de este análisis se generan insights que permiten apoyar la toma de decisiones y mejorar la estrategia de productos y planes de la empresa.

# datasets utilizados 
El proyecto utiliza tres datasets principales:
users: información de los clientes (edad, ciudad, plan, fechas de registro y cancelación).
usage: registros de uso de servicios (llamadas y mensajes).
plans: información de los planes disponibles.

# etapas de analisis realizados
Carga y exploración inicial de los datos.
Identificación de valores nulos y datos faltantes.
Limpieza datos 
Detección y tratamiento de sentinels (valores inválidos).
Conversión y validación de fechas.
Análisis estadístico descriptivo.
Visualización de datos (histogramas y boxplots).
Detección de outliers usando el método IQR.
Creación de variables agregadas por usuario.
Segmentación de clientes por edad y nivel de uso.
Elaboración de conclusiones y recomendaciones.
